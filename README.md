## Inicio rapido
Se necesita tener instalado [node](https://nodejs.org/en/about) luego instalar el
proyecto, clonarlo, como prfiera... dentro del proyecto usar el siguente comando:

```bash
  npm install
```
luego corra el programa:

```bash
  npm run dev
```
